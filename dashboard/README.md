# Dashboard Power BI

## Archivo

- `TP3_AD.pbix`
- Paginas detectadas: `General`, `Clusters`, `Detalle Barrio`, `Detalle Propiedad` y `Tooltip M2`.

El tablero permite recorrer el embudo de negocio desde una vista general hasta el detalle de una propiedad. Incluye filtros, mapas, comparacion de precio por m2 y composicion del indice de oportunidad.

## Guia opcional: publicar en Power BI Service

El tablero ya esta publicado y navegable online (ver seccion "Enlace online"). Estos son los pasos seguidos para republicarlo o actualizarlo:

1. Abrir `TP3_AD.pbix` en Power BI Desktop.
2. Verificar que las fuentes apunten a los archivos de `data/processed`.
3. Actualizar los datos y revisar que no existan errores.
4. Publicar en Power BI Service.
5. Configurar credenciales y actualizacion cuando corresponda.
6. Obtener el enlace compartible permitido por la institucion.
7. Incorporar el enlace en el README principal.

## Control previo a entrega

- Titulos y unidades consistentes con el README.
- Precio expresado en USD y precio por m2 claramente identificado.
- Filtros de barrio, cluster, estado y tipo de unidad operativos.
- Drill-through entre barrio y propiedad.
- Ranking interpretable como priorizacion, no como ROI.
- Advertencia visible sobre precios publicados y validaciones pendientes.
- Sin enlaces locales rotos ni visuales con errores.
